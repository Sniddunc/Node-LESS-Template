Basic LESS Template
by Duncan Snider

The LESS source directory is called less and is located in the root directory

The folder run by the server is called public and is located in the root directory

The file the LESS source is compiled to is called style.css and is located in /public/css

Server Commands

start

Starts the development server and watches for LESS changes

less

Compiles LESS into CSS

less:watch

Compiles LESS into CSS and watches for changes

server

Starts the development server
